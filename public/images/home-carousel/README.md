# Home Carousel Images

Place the homepage carousel photos in this folder.

Recommended naming:
- slide-01.jpg
- slide-02.jpg
- slide-03.jpg
- slide-04.jpg
- slide-05.jpg

Then update `HERO_SLIDESHOW_IMAGES` in `constants.ts` to point to:
- /images/home-carousel/slide-01.jpg
- /images/home-carousel/slide-02.jpg
- /images/home-carousel/slide-03.jpg
- /images/home-carousel/slide-04.jpg
- /images/home-carousel/slide-05.jpg
